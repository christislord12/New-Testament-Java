# New-Testament-Java
Application to read the New Testament and the Book Of Psalms created with Java

Due to Github only supporting files under 100MB, the JAR file had to be compressed into multiple zip files. You can download the JAR file here: [NewTestamentPsalms.jar](https://github.com/christislord12/New-Testament-Java/releases/download/1.0/NewTestamentPsalms.jar)
